# Betânia Bot — Agente IA para WhatsApp

Agente IA da Igreja Betânia integrado com Evolution API e Claude (Anthropic).

## Variáveis de ambiente necessárias no Railway:

- `ANTHROPIC_API_KEY` — Sua chave da Anthropic (sk-ant-...)
- `EVOLUTION_API_URL` — URL do servidor Evolution API
- `EVOLUTION_API_KEY` — Senha da Evolution API (betania2026)
- `INSTANCE_NAME` — Nome da instância (betania)
